# IpSearch

PC방 로그 조회를 위한 API 서버입니다.  
IP 기준으로 접속 로그를 조회하고, 관리자 또는 운영자가 필요한 로그 정보를 빠르게 확인할 수 있도록 구성한 프로젝트입니다.

## 📌 프로젝트 개요

이 프로젝트는 PC방 환경에서 발생하는 접속 로그를 조회하기 위한 백엔드 API 서버입니다.

주요 목적은 다음과 같습니다.

- IP 기준 로그 조회
- PC방 사용자 접속 이력 확인
- 관리자용 로그 검색 API 제공
- 서버 기반 로그 데이터 관리

## 🛠 기술 스택

- C#
- ASP.NET Core
- Entity Framework Core 또는 Dapper
- MySQL / MariaDB
- REST API
- Swagger
